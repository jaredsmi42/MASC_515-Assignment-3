Using a externally provided Mircogpt code this assignment focuses on using that code as the base and with the assistance of AI editing the based code to incorporate other machine learning algorithms.

Gaussian Error Learning units: Functions by scaling each input smoothly based upon its magnitude using a gaussian probability curve opposed to making a hard cutoff point. large inputs and positive are mostly unchanged by this however smaller or negative inputs are gradually reduced instead of xoered out leading to the smoother presentation of data.

LoRA: This algorithm works by freezing the original large model's weights and instead learning small low rank- matrices that modify each layer's behavior, rather than retraining the entire network. This behvior can approximate how the whole model would change through utilzing fewer parameters thus providing fine tuning at a fraction of the computational expense.

RoPE Embedding: Rotary Position Embedding (RoPE) works by encoding positional information as rotations applied to token embeddings, so each token’s representation is transformed based on its position in the sequence. This allows the model to naturally capture relative relationships between tokens (how far apart they are) directly within the attention mechanism, rather than relying only on absolute position values. As a result, RoPE improves the model’s ability to handle long sequences and varying input lengths while maintaining meaningful dependencies between tokens.
